Las migraciones de esta carpeta no tienen marca temporal, sirven como referencia.

Las migraciones deben crearse en un orden que no genere confictos y luego debe
sustituirse cada contenido de *_create_xxx_table.php por el de CreateXxxTable.php:

php artisan make:migration create_cursos_table
php artisan make:migration create_docentes_table
php artisan make:migration create_encuestas_table
php artisan make:migration create_password_resets_table
php artisan make:migration create_preguntas_table
php artisan make:migration create_realizadas_table
php artisan make:migration create_respuestas_table
php artisan make:migration create_users_table

Nunca eliminar la carpeta migrations, (pude quedar vacia, pero debe existir).

Si artisan sigue queriendo leer migraciones viejas,
ejecutar un "composer dump-autoload" (quizas tambien un "git add -u .").

En la carpeta migrations solo deberia de existir migraciones.
